# ExpressJS.com的中文版

  [ExpressJS.com](http://expressjs.jser.us)的中文版本。 


## 运行

安装:

```
$ npm install -g serve
$ npm install
$ make
$ serve . &
$ open http://localhost:3000
```

如果有新的改变，重新编译一下:

```
$ make clean;make docs
```
